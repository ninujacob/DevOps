### AWS Authetication
- Create a programatic access(AWS_ACCESS_KEY_ID & AWS_ACCESS_KEY_SECRET)
- Export variables
  ```
  export AWS_ACCESS_KEY_ID=<key_id>
  ```
  ```
  export AWS_SECRET_ACCESS_KEY=<secret>
  ```
### Validate Access
- Verify the access configuration is valid or not
  ```
  aws s3 ls
  ```
